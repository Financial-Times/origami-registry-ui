What is an Origami component?

An Origami component is a collection of styles and functionality. It provides reusable HTML, SCSS and JavaScript for a multitude of different products under the FT umbrella.

When is the component registry updated?

The component registry is updated whenever a new tag is pushed to a repository that has an origami.json file in it. If your repository meets the following criteria then it should be ingested within 5 minutes:

  • It lives in the Financial-Times GitHub organisation
  • It contains a valid origami.json file (see the spec for more information)
  • It has at least one git tag which complies with semantic versioning
