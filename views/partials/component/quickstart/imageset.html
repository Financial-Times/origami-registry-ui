Using the Imageset

Image set images can be loaded directly using the Image Service. This scheme for this image set is: {{@root.component.imageSetScheme}}

To load an image from this image set, use the following URL for the Image Service:

https://www.ft.com/__origami/service/image/v2/images/raw/{{@root.component.imageSetScheme}}:{image-name}?source={your-source-here}
